LoveGood
===
项目介绍（如果对你有用，请给一个star！）
-
* 整个项目是以保健食品为服务中心的专题网站。

功能介绍
-
* 首页功能：导航模块（自定义导航设置），广告模块（自定的轮播设置），二级菜单（自定义）
* 列表功能：分类模块（自定义分类设置），搜索引擎（自定义搜索设置）
* 关于我们详情页功能：公司简介，公司说明等。
* 资讯中心： 正文介绍，广告等功能。
